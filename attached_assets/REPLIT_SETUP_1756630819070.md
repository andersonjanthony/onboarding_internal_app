## Step 2: Configuration

### Automatic Setup
Replit should automatically:
- Detect `package.json` and install dependencies
- Set up the run command (`pnpm dev`)
- Configure the preview window

### Manual Steps (if needed)
1. **Install Dependencies**:
   ```bash
   pnpm install
   ```

2. **Set Run Command** (in `.replit` file):
   ```
   run = "pnpm dev"
   ```

3. **Environment Variables** (optional):
   - Go to "Secrets" tab in Replit
   - Add `VITE_SLACK_WEBHOOK` and `VITE_N8N_WEBHOOK` if needed

## Step 3: Run the App

1. Click the "Run" button in Replit
2. Wait for the build to complete
3. Open the preview window
4. You should see the SecureForce onboarding app!
